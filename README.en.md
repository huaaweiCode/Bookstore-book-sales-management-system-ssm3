# 书店图书销售管理系统SSM3

#### Description
书店图书销售管理系统3具有两种个角色，分别为管理员和用户，具体功能如下：

管理员：书籍的增删改查、书籍类型的增删改查、用户的增删改查、订单审核、订单详情查看等功能

用户：书籍的模糊查询、购买数据、购物车、结算、注册登录等功能
